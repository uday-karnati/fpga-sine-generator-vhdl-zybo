# Configurable Sine Wave Generator — Zybo FPGA (VHDL)

A fully hardware-implemented sine wave generator on the Zybo FPGA 
board. Sine wave parameters (frequency, amplitude) are loaded from 
a PC via UART, stored in a register file, and the resulting 
waveform is displayed live on a VGA monitor.

University project — Ravensburg-Weingarten University of Applied 
Sciences (RWU), 2024

## What it does
- Receives sine wave parameters from a PC over UART (115200 baud)
- Stores parameters in an FSM-controlled register file
- Generates a configurable sine wave using a quadrant-based 
  counter approach
- Displays the sine waveform in real time on a VGA monitor (white 
  pixel trace on black background)
- Transmits generated sine values back to PC via UART TX

## System architecture
```
PC (serial terminal)
        │ UART RX
        ▼
  Register File (FSM)
        │ parameters
        ▼
  Sine Counter (FSM)
        │ sine values
        ├──────────────────┐
        ▼                  ▼
   UART TX            VGA Display
  (to PC)          (VGA monitor)
```

## Hardware
- Board: Digilent Zybo (Xilinx Zynq-7000)
- Tools: Vivado, ModelSIM
- Language: VHDL
- Clock: 125 MHz input → divided to 25 MHz for VGA pixel clock
- UART: 115200 baud, 8-bit, no parity

## How the sine generator works
The sine counter uses a 4-quadrant FSM approach:
1. Quadrant 1 — reads register addresses 0→7 (rising)
2. Quadrant 2 — reads register addresses 7→0 (falling)
3. Quadrant 3 — reads register addresses 0→7 (negative rising)
4. Quadrant 4 — reads register addresses 7→0 (negative falling)

Each register holds one sine coefficient loaded from the PC.
The FSM cycles through all 4 quadrants to reconstruct a full 
sine period.

## VGA display
- Resolution: 640×480 @ 60Hz
- Pixel clock: 25 MHz (generated by clock divider)
- H-sync and V-sync generated by separate counters
- Sine value mapped to horizontal pixel position
- White pixel drawn at sine amplitude, black everywhere else

## Register FSM states
- `RST_ST` — idle, waiting for input
- `WR_DATA_ST` — writing received UART byte to register
- `R_DATA_ST` — reading register value for sine counter
